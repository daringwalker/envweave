# EnvWeave 0.1.0-alpha.2

这是针对 Linux Wayland 图形兼容性的 Alpha 修复版本。

## 修复内容

- Linux Wayland 会话启动时默认关闭 WebKitGTK 的 DMABUF 快速渲染路径，规避部分 Arch Linux、KDE Plasma Wayland 和 NVIDIA 驱动组合出现的空白窗口。
- 保留用户显式设置的 `WEBKIT_DISABLE_DMABUF_RENDERER` 与 `WEBKIT_DISABLE_COMPOSITING_MODE`，方便诊断或恢复默认渲染策略。
- Linux 虚拟机发布验收会直接启动最终 AppImage，并检查窗口内是否渲染出有效内容。

## 安装与校验

下载对应平台安装包后，使用 Release 中的 `SHA256SUMS.txt` 校验文件。Linux AppImage 首次运行前需要添加执行权限：

```bash
chmod +x EnvWeave_0.1.0-alpha.2_linux_x86_64.AppImage
./EnvWeave_0.1.0-alpha.2_linux_x86_64.AppImage
```

本版本仍是开发者预览版。迁移重要配置前，请保留独立备份。
