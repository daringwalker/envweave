# EnvWeave 0.1.0-alpha.4

本版本重点提升新机器迁移的安全性、Git 冲突恢复能力和桌面界面可读性。

## 主要变化

- 目录恢复支持 `replace`、`merge` 和 `keep-existing`。新添加的目录默认使用 `merge`，保留新系统或应用生成的本机独有内容。
- 恢复计划会列出新增、覆盖、保留和删除路径；任何删除操作都需要用户明确勾选确认。
- 排除路径在采集时不会进入新快照，恢复时也不会覆盖或删除本机对应内容。
- Git pull/rebase 发生冲突后，可在同步页查看冲突文件、重新检查、继续或安全中止。
- 浅色、深色和 Monaco 编辑器主题已经统一，并使用完整模拟业务数据验证可读性。
- 智能扫描的配置文件选择与右侧内容预览联动更加稳定。

## 安装与校验

Linux x86_64 用户可下载 AppImage 或 Debian 包：

```bash
chmod +x EnvWeave_0.1.0-alpha.4_linux_x86_64.AppImage
./EnvWeave_0.1.0-alpha.4_linux_x86_64.AppImage
```

macOS 用户应选择与 Apple Silicon 或 Intel 架构对应的 DMG。Alpha 版 macOS 产物未承诺完成 Developer ID 公证，只适用于受控测试。

下载后使用 Release 中的 `SHA256SUMS.txt` 校验文件。本版本仍是开发者预览版，迁移重要配置前请保留独立备份；系统级配置仍不会直接写回，秘密文件也不应放入普通 Git 仓库。
