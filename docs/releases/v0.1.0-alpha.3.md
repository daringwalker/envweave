# EnvWeave 0.1.0-alpha.3

这是针对 Arch Linux、KDE Plasma Wayland 环境中 AppImage 空白窗口的兼容性修复版本。

## 修复内容

- AppImage 的 `AppRun` 会识别 Arch、Manjaro、EndeavourOS 等 Arch 系发行版，并在启动 WebKitGTK 前预加载系统 `libwayland-client.so.0`。
- 保留 Linux Wayland 的 WebKitGTK DMABUF 兼容处理，同时尊重用户显式设置的 WebKit 环境变量。
- 发布验收现在会把 Ubuntu 构建的最终 AppImage 复制到当前 Arch Linux 虚拟机直接启动，并检查界面是否真正渲染。

## 安装与校验

下载 Linux x86_64 AppImage 后执行：

```bash
chmod +x EnvWeave_0.1.0-alpha.3_linux_x86_64.AppImage
./EnvWeave_0.1.0-alpha.3_linux_x86_64.AppImage
```

使用 Release 中的 `SHA256SUMS.txt` 校验下载内容。本版本仍是开发者预览版，迁移重要配置前请保留独立备份。
