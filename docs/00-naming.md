# 产品命名

## 正式名称

**EnvWeave**

含义：将配置文件、软件包、平台规则与恢复流程编织成一个可迁移、可重建的系统环境。

## 命名规范

- 产品显示名：`EnvWeave`
- 可执行文件与未来 CLI：`envweave`
- Rust crate 前缀：`envweave-*`
- 仓库主清单：`envweave.toml`
- 软件包清单：`packages.toml`
- 应用标识：`io.envweave.desktop`
- 默认配置目录：Linux `~/.config/envweave`；macOS 使用应用支持目录中的 `EnvWeave`

## 品牌语言

- 英文短句：`Weave your environment. Rebuild anywhere.`
- 中文短句：`编织你的环境，随处重新开始。`

## 使用约束

- 正文始终写作 `EnvWeave`，不使用 `Envweave` 或 `ENVWEAVE`。
- 代码、路径和命令使用小写 `envweave`。
- 不把产品称为纯 dotfile manager，因为软件包迁移也是核心能力。
- 正式公开发布前仍需完成目标市场的商标、域名和应用商店名称核查。
